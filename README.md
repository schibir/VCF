# VCF
VCF solver

### Hot keys:

* Ctrl + N -- clear board
* Ctrl + Z -- Undo
* Ctrl + Y -- Redo
* Home -- Undo all
* End -- Redo all
* Ctrl + I -- switch enable/disable engine
* Arrows -- Move position
* Ctrl + S -- save position
* Ctrl + O -- Open position
* Ctrl + V -- Past position string from clipboard (like h8g7g9i7f9h9)
* Ctrl + C -- Copy position string to clipboard
